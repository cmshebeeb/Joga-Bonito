🗂️ How This Works

All data lives in data.js. You edit it, push to GitHub, and the website updates for everyone worldwide in ~30 seconds — permanently, for free, forever.

No database. No server. No cost. Just a text file.

📁 Your Repo Folder Structure

Joga-Bonito/ ├── index.html ← website (rarely need to touch) ├── data.js ← YOUR DATABASE ← edit this └── images/ ├── logo.png ├── teams/ │ └── fc-kerala.png ├── players/ │ └── rahul.jpg ├── cards/ │ └── rahul-card.jpg ← 9:16 portrait └── news/ └── matchday.jpg

⚡ Every Update — 3 Commands

  1. Open VS Code → edit data.js
  2. Drop any new images into the images/ folder
  3. Open terminal (Ctrl+`) and run:
git add . git commit -m "Added match result: FC Kerala 2-1 Mallus United" git push

✅ Live in ~30 seconds for everyone.

➕ Add a Match Result

In data.js, find the matches array and add:

{ id: 1001, homeId: 1, awayId: 2, date: "2026-04-15", label: "Group Stage", homeScore: 2, awayScore: 1, goals: [ { playerId: 101, teamId: 1, og: false }, { playerId: 101, teamId: 1, og: false }, { playerId: 103, teamId: 1, og: true }, // own goal ], cards: [ { playerId: 102, type: "yellow" }, ], isFinal: false },

Points table and stats update automatically.

👤 Add a Player

{ id: 101, name: "Rahul M", teamId: 1, // matches team id position: "Attacker", // Goalkeeper|Defender|Midfielder|Attacker captain: true, // only ONE true per team photo: "images/players/rahul.jpg", card: "images/cards/rahul-card.jpg", // 9:16 ratio order: 0 // display order (0,1,2,...) },

🖼️ Add Images

  1. Rename file simply: rahul.jpg, fc-kerala.png
  2. Resize to under 300KB at squoosh.app
  3. Drag into the correct images/ subfolder in VS Code
  4. In data.js set: photo: "images/players/rahul.jpg"
  5. git add, commit, push

🏆 Set the Winner

winner: { teamId: 1, // winning team's id showCelebration: true // shows banner + confetti },

Also set isFinal: true on the championship match.

📋 Publish Fan Reviews (from Google Sheet)

  1. Open your Google Sheet (connected via Formspree — see Form Setup tab)
  2. Copy a fan's answers
  3. Add to the reviews array in data.js (template is in data.js)
  4. git push — analytics appear instantly for everyone

📬 One-Time Setup: Fan Form → Google Sheet

Do this once. Takes about 10 minutes.

  1. 1Go to formspree.io → Sign up free (use your Google account)
  2. 2Click "+ New Form" → name it "Joga Bonito Review" → Create
  3. 3Copy the Form ID (looks like xpwzabcd)
  4. 4Open data.js and paste it:
    formspreeId: "xpwzabcd"
  5. 5In Formspree → your form → Integrations tab → Google Sheets → Connect → choose or create a Google Sheet
  6. 6git push your updated data.js
  7. 7Test: submit one review on the website → check your Google Sheet for the row

✅ What You Get

Every fan submission auto-appears as a new row in your Google Sheet with: name, formation, squad picks, all award votes, team rankings, comments, and timestamp. The data is yours forever in Google Drive.

💰 Cost?

Formspree free plan: 50 submissions/month. That's more than enough for a tournament of 30–50 players. GitHub Pages: free forever. Total cost: ₹0.
