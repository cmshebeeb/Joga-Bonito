🗂️ How This Works

All data lives in data.js. Edit it, push to GitHub, and the website updates in ~30 seconds — free, forever.

📁 Repo Structure

Joga-Bonito/ ├── index.html ← website structure ├── data.js ← YOUR DATABASE ← edit this ├── script.js ← all JavaScript ├── styles.css ← all styles └── images/ ├── logo.png ├── teams/ ← team logos ├── players/ ← player photos ├── cards/ ← 9:16 player cards ├── events/ ← event carousel photos └── sponsors/ ← sponsor photos

⚡ Every Update — 3 Commands

  1. Edit data.js in VS Code
  2. Drop images into images/ subfolder
  3. Run in terminal:
git add . git commit -m "Updated match result" git push

✅ Live in ~30 seconds.

👥 Add Login Users

Players are auto-added from the players array. For sponsors and fans, edit the users array in data.js:

users: [ // Sponsors: { username: "reliance", password: "pass123", usertype: "s", displayName: "Reliance Foundation" }, // Fans: { username: "arun", password: "fan001", usertype: "f", displayName: "Arun K" }, ],

➕ Add a Match Result (with goals & cards)

{ id: 1001, homeId: 1, awayId: 2, date: "2026-04-15", time: "16:30", label: "Group Stage", homeScore: 2, awayScore: 1, goals: [ { playerId: 102, teamId: 1, og: false, assistId: 103 }, { playerId: 103, teamId: 1, og: false, assistId: 0 }, { playerId: 111, teamId: 1, og: true, assistId: 0 }, // own goal ], cards: [ { playerId: 108, type: "yellow" }, { playerId: 114, type: "red" }, ], isFinal: false },

🏆 Set the Winner

winner: { teamId: 1, // winning team's id showCelebration: true // shows banner + confetti },

Also set isFinal: true on the championship match.

📋 Publish Fan Reviews

  1. Fan submits form → lands in your Google Sheet (via Formspree)
  2. Copy their row into the reviews array in data.js
  3. git push — analytics update instantly for everyone

📬 One-Time Setup: Fan Form → Google Sheet

Do this once. Takes about 10 minutes.

  1. 1Go to formspree.io → Sign up free
  2. 2Click "+ New Form" → name it "Joga Bonito Review" → Create
  3. 3Copy the Form ID (e.g. xpwzabcd)
  4. 4Open data.js and paste:
    formspreeId: "xpwzabcd"
  5. 5Formspree → your form → Integrations → Google Sheets → Connect
  6. 6git push your updated data.js
  7. 7Test: submit one review → check Google Sheet

💰 Cost?

Formspree free plan: 50 submissions/month. GitHub Pages: free forever. Total: ₹0.
